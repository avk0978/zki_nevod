# Nevod — контекст проекта для Claude Code

## Что это

Nevod — децентрализованный защищённый мессенджер для личного использования.
Сеть равноправных узлов (P2P), без центрального сервера.
Рабочая папка: D:\AVK\Nevod

## Ключевые решения (приняты)

- Транспорт: WebSocket over TLS
- Топология: равноправные узлы (не иерархия)
- Регистрация нового узла: 50% консенсус активных узлов
- Удаление узла: 72 часа недоступен → автоудаление из сети
- Ячейка = пользователь, идентичность = Ed25519 keypair (портативна)
- Два типа узлов: постоянный (50% консенсус) и эфемерный (приглашение home-узла)
- При падении узла данные теряются — конфиденциальность важнее сохранности
- Буферизация: узел хранит сообщения для оффлайн-адресатов, TTL 72ч
- Presence: home-узел знает где сейчас находится ячейка
- E2E шифрование: узлы видят только зашифрованные blob-ы, не содержимое
- Аутентификация: ZKP (протокол Шнорра) + MFA (TOTP)
- Крипто: PyNaCl (Ed25519, X25519, ChaCha20)
- Протокол сообщений: MessagePack
- Node table sync: Gossip protocol (30 сек)
- Язык: Python (asyncio)
- Хранилище: SQLite
